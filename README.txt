Requirements:
- Python 3.10.7
- Python packages
  - pandas
  - seaborn
  - streamlit
  - plotly
  - leafmap
  - geopandas

Getting Started:
1) cd to project directory
2) Run data.py to generate all required csv files for streamlit
3) Run visualization_main.py with either commands:
- "python streamlit run visualization_m.py"
- "py -m streamlit run visualization_main.py"
